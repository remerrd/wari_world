Welcome to Wari-World!

This is a fantasy based text adventure, the players make choices in different situations to lead to different endings.

The world is expanding, constantly growing based on players.

How It Works

Playing the Game:

  1. Read the situation
  2. Look at avaliable options
  3. Type in text with one of the option words
  4. If you get to a point that no one has been to, you start to create the world
